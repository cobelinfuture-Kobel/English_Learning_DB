## 1. Allowed Pattern Families

### A. Literal Comprehension

Purpose:
Check direct understanding of visible learner-facing text.

Examples:

```text
who
what
where
when
yes_no
true_false
```

### B. Vocabulary-in-Context

Purpose:
Check meaning from local sentence context.

Examples:

```text
word_meaning
picture_word_match
simple_synonym_choice
```

### C. Sentence Order and Sequencing

Purpose:
Check simple event order.

Examples:

```text
first_next_last
sentence_order
picture_sequence_match
```

### D. Reference and Detail Match

Purpose:
Match a prompt to a detail in the private homework display payload.

Examples:

```text
detail_match
character_match
place_match
action_match
```
